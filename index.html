Analyzing Flat SPR Dissociation Phases

Surface Plasmon Resonance (SPR) instruments measure molecular interactions. Data may exhibit a flat dissociation phase when molecules bind with high affinity. Kinetic analysis software frequently attempts to fit a mathematical model to this flat section. This results in the quantification of baseline instrument noise rather than the true biological interaction. Best practices dictate defining the lowest measurable limit of the instrument instead of reporting non-physical absolute values.

A minimum signal decay of 5% is generally required to confirm an observable dissociation event. If the signal decays by less than 5% over the observed window, the true dissociation rate ($k_d$) cannot be accurately determined. An observable limit should be calculated for these scenarios.

1. The Limit Calculator

The minimum observable dissociation rate ($k_{d\_min}$) depends on the duration of the dissociation phase ($t_d$). This utilizes a first-order exponential decay relationship.

$$k_{d\_min}=\frac{-\ln(0.95)}{t_d}$$

Reporting Protocols

It is recommended practice to report flat dissociation data as limits rather than distinct numerical values. The $k_d$ is recorded as an inequality. This denotes it is less than the calculated minimum observable limit.

The equilibrium dissociation constant ($K_D$) represents the ratio of the off-rate to the on-rate ($k_a$).

$$K_D=\frac{k_d}{k_a}$$

Calculate the $K_D$ limit using the $k_d$ limit and the measured $k_a$. The experimental methodology section should explicitly state that the minimum dissociation rate was determined utilizing a 5% signal loss threshold over the measured dissociation duration.

Reporting Example:

If the mathematical fit yields a non-physical $k_d$ of $1.00 \times 10^{-6}$ s$^{-1}$ for a 600s dissociation phase, do not report it. Instead, utilize the calculated limit ($8.5 \times 10^{-5}$ s$^{-1}$). Assuming an accurately measured $k_a$ of $9.16 \times 10^{4}$ M$^{-1}$s$^{-1}$, the final reported parameters should look like this:

$$k_a = 9.16 \times 10^{4} \text{ M}^{-1}\text{s}^{-1}$$ $$k_d < 8.50 \times 10^{-5} \text{ s}^{-1}$$ $$K_D < 9.28 \times 10^{-10} \text{ M}$$

2. SPR Kinetic Simulator

Enter a target affinity ($K_D$) and select a scenario. The simulation automatically adjusts the concentration for slower kinetics. This ensures the interaction reaches sufficient signal amplitude for accurate visual comparison.

Implementation in the Nicosystem

You can force the software to recognize this physical limit by setting the parameter manually.

  1. Go to Model and Accommodation.
  2. Set the $k_d$ to constant using the limit value which corresponds to your dissociation time.
  3. Select Apply Locally to run the fit with this boundary.

Handling Replicates and Baseline Variability

When evaluating multiple replicates, slight variations in the baseline can occur. This is a common aspect of instrument performance. However, this variability can complicate analysis if the algorithm attempts to fit these minor fluctuations as true dissociation events. Ensure proper double-referencing is applied to remove systematic background noise. Buffer Blanks are available on SW ver 2.6.0+ to assist with this subtraction.

If the sensorgrams remain largely flat but show minor differences, do not average the separate $k_d$ values generated by the software. Evaluate the total signal change over the dissociation period for each replicate relative to the 5% threshold.
